# SWIMS: Supplier-Warehouse Inventory Management System

## Description
<p>This system manages inventory for a company that receives products 
  from suppliers and stores them in warehouses in multiple locations. 
  It keeps records of suppliers, products, warehouses, inventory levels, 
  and stock movements so that changes in product, location, and stock 
  can be tracked. The system is limited to inventory management and does
  not include customers, billings, or orders.</p>
